# SIAuto Desktop

## Ngejalankan Aplikasi
```bash
git clone https://github.com/oktoala/siauto-desktop.git
cd siauto-desktop
npm install
npm run start
```

## Ngebuat Aplikasi
```bash
npm run package
```
